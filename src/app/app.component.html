Cómo Funciona JWT

Los JSON Web Tokens (JWT) son un método seguro y compacto para la transmisión de información entre partes como un objeto JSON. Esta información puede ser verificada y confiada porque está firmada digitalmente.

  • Autenticación: El usuario inicia sesión y el servidor emite un JWT.
  • Transmisión: El cliente envía el JWT en cada solicitud protegida.
  • Verificación: El servidor valida el token sin consultar una base de datos de sesiones.
